Name exports from Shopify product CSV

Upload a Shopify product export (same columns as Handle, Type, Option1 Value for color, Variant SKU). Choose the product, then Apply names to export fields: the export folder becomes the handle, and each WebP name is the handle, then that color’s Variant SKU, then the swatch name (dash-separated). Variant SKU uses the first size row per color. Your export swatch name must match the CSV color (e.g. Navy ↔ Option1 Value Navy). The ZIP download lists all WebPs at the top level of the archive.

Advanced — fine-tune mask & image quality

Masking and recoloring run automatically after upload. Use this panel to fix stubborn edges, same-color backgrounds, or to adjust fabric detail and edge cleanup.

Higher removes more backdrop. White-on-white flat-lays with soft drop shadows are detected automatically (sharper silhouette + less rim wash). If fabric vanishes, lower sensitivity and Refresh cutout.

Same-color garment vs background
